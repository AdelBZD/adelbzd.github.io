Master’s Internship

Evaluation and Optimization of LLMs for HPC

CERFACS (COOP Team) — Toulouse

During my Master’s internship, I work in an applied research environment focused on the use of AI in HPC. The main objective is to evaluate and improve large language models for scientific code generation.

  • Comparative evaluation of around 40 LLMs for Fortran code generation
  • Work around OpenMP / OpenACC-oriented scientific programming tasks
  • Analysis of a Docker-based pipeline combining local model execution and benchmark evaluation
  • Detailed study of benchmark structure: prompt generation, compilation, execution, validation, and failure analysis
  • Reasoning about automated evaluation, guided correction, and reliability of generated code
  • Experiments in a GPU-equipped environment
Master 1 Project

Kubernetes, MLOps, and Distributed Training

University of Perpignan Via Domitia

This project combined deep learning, deployment, orchestration, and observability in a complete technical workflow.

  • Setup of a Kubernetes cluster with Docker, Calico, Helm, MetalLB, and shared NFS storage
  • Development of an LSTM-based text autocompletion system in Python
  • Sequential and distributed training using Horovod and Kubeflow MPIJob
  • Deployment of trained models on Kubernetes
  • Monitoring of performance and predictions with Prometheus and Grafana
  • Development of an interactive web interface for demonstration purposes
Previous Master’s Project

Predictive Resource Management in Kubernetes

University of Bouira

This project focused on predicting the resource consumption of a microservice in Kubernetes using an encoder-decoder LSTM model.

  • Deployment of a Kubernetes cluster with NFS-based storage
  • Training data construction from CPU load generation and metric collection
  • Time-series data storage with InfluxDB
  • Implementation of predictive autoscaling strategies
  • Vertical scaling of resources and horizontal scaling through replica adjustment
  • Visualization and monitoring with Grafana
